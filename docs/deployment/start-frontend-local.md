# Démarrer le Frontend Localement

## Installation
```bash
cd skybot-inbox-ui
npm install
```

## Démarrer en mode dev
```bash
npm run dev
```

Ensuite va sur: **http://localhost:3000**

Le frontend se connectera automatiquement à ton backend Render:
- Backend: https://skybot-inbox.onrender.com/api
- Frontend: http://localhost:3000
